¿Cómo defines una función? 
        
  
            Una función de JavaScript se define con la 
            function palabra clave, seguida de un nombre,
            seguido de paréntesis ().
            Los nombres de funciones pueden contener 
            letras, dígitos, subrayados y signos de 
            dólar (las mismas reglas que las variables).
            Los paréntesis pueden incluir nombres 
            de parámetros separados por comas:
            ( parámetro1, parámetro2,... )
            El código a ejecutar, por la función,
